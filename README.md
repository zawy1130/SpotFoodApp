# SpotFood Mobile App

SpotFood is a mobile application designed to help users discover food locations, restaurants, cafes, and local points of interest.

The application provides location-based food discovery with multilingual content support and media integration.

---

## Features

- Browse nearby food locations
- View POI details and descriptions
- Category-based filtering
- Multilingual translation support
- Audio content support
- Real-time data synchronization with Admin CMS

---

## Tech Stack

- Android (Java/Kotlin)
- Retrofit
- RESTful API
- SQL Server
- ASP.NET Core Backend
